gen_cache: generic cache module.

This module defines the gen_cache behaviour.
Required callback functions: init/1, reset/1, handle_lookup/3, handle_touch/3, handle_insert/3, handle_update/3, handle_replace/3.

Authors: James Ruan (ruanbeihong@gmail.com).
